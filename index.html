Final class project for digital electronics class in the Music Technology program at New York University

Project was designed using Arduino and Processing. Arduino controller is designed to trigger MIDI samples in Ableton Live. Project is a four step sequencer, with each channel represented by a different color (show by the LEDs on controller).

When a specific channel is selected, the Processing sketch will change to the color of the selected channel. Each time the kick drum (channel 1) is triggered, the circles in the sketch will increase in size. After a certain size is reached, circles return to normal size and keep increasing from there.

Final class project for analog electronics class in the Music Technology program at New York University.

Drone synthesizer consisting of six droning oscillators that can be controlled with their own pitch knobs. The instrument also has a master volume control, as well as a button to momentarily lower the volume of all oscillators at once. This aspect allows for more performance control of the instrument.

Senior Capstone project at New York University, Music Technology program.

Custom designed, Pure Data based FM synthesizer. Runs headless with Patchbox OS on Raspberry Pi, hardware control done through Arduino. Use of proximity sensor data to manipulate synthesizer parameters, as well as potentiometers, buttons, and switches. Arduino interfaces with Raspberry Pi/Pure Data through USB MIDI protocols.
